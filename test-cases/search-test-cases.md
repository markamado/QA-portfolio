## Тест-кейсы поиска (SauceDemo)

### TК-001: Поиск существующего товара по полному названию

**Предусловие:** пользователь находится на странице каталога https://sauce-demo.myshopify.com/collections/all

**Шаги:**
1. Ввести в поле поиска "Black heels"
2. Нажать Enter / кнопку поиска

**Ожидаемый результат:**
В результатах отображается товар "Black heels"

### TК-002: Поиск товара по частичному совпадению

**Предусловие:** пользователь находится на странице каталога

**Шаги:**
1. Ввести в поле поиска "Black"
2. Нажать Enter / кнопку поиска

**Ожидаемый результат:**
Отображаются товары, содержащие "Black" в названии

### TК-003: Поиск несуществующего товара

**Предусловие:** пользователь находится на странице каталога

**Шаги:**
1. Ввести в поле поиска "Random123"
2. Нажать Enter / кнопку поиска

**Ожидаемый результат:**
Отображается сообщение об отсутствии результатов

### TК-004: Поиск с учетом регистра

**Предусловие:** пользователь находится на странице каталога

**Шаги:**
1. Ввести в поле поиска "black heels"
2. Нажать Enter / кнопку поиска

**Ожидаемый результат:**
Отображается товар "Black heels" (поиск не зависит от регистра)

### TК-005: Поиск с пустым запросом

**Предусловие:** пользователь находится на странице каталога

**Шаги:**
1. Оставить поле поиска пустым
2. Нажать Enter / кнопку поиска

**Ожидаемый результат:**
Страница не ломается, отображается либо полный каталог, либо ничего не происходит

### TК-006: Поиск со спецсимволами

**Предусловие:** пользователь находится на странице каталога

**Шаги:**
1. Ввести в поле поиска "@#$%"
2. Нажать Enter / кнопку поиска

**Ожидаемый результат:**
Отображается сообщение об отсутствии результатов, страница работает корректно

### TК-007: Переход в карточку товара из результатов поиска

**Предусловие:** выполнен поиск, отображаются результаты

**Шаги:**
1. Нажать на любой товар из результатов поиска

**Ожидаемый результат:**
Открывается карточка выбранного товара

### TК-008: Ввод длинного поискового запроса

**Предусловие:** пользователь находится на странице каталога

**Шаги:**
1. Ввести в поле поиска строку длиной более 100 символов
2. Нажать Enter

**Ожидаемый результат:**
Поиск обрабатывается корректно, верстка страницы не ломается

### TК-009: Быстрое изменение поискового запроса

**Предусловие:** пользователь находится на странице каталога

**Шаги:**
1. Ввести "Black"
2. Быстро удалить текст
3. Ввести "Jacket"
4. Нажать Enter

**Ожидаемый результат:**
Отображаются корректные результаты для "Jacket", без ошибок

### TК-010: Очистка поискового запроса

**Предусловие:** выполнен поиск

**Шаги:**
1. Очистить поле поиска
2. Нажать Enter / обновить страницу

**Ожидаемый результат:**
Отображается полный список товаров
