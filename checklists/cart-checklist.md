# Корзина (SauceDemo)

## Страница корзины
- [ ] Отображаются добавленные товары
- [ ] У каждого товара отображается название
- [ ] У каждого товара отображается цена
- [ ] Отображается количество товаров
- [ ] Отображается итоговая сумма заказа

## Работа с количеством
- [ ] Увеличение количества товара работает корректно
- [ ] Уменьшение количества товара работает корректно
- [ ] При изменении количества пересчитывается итоговая сумма

## Удаление товара
- [ ] Кнопка "X" удаляет товар из корзины
- [ ] После удаления товар исчезает из корзины
- [ ] После удаления пересчитывается итоговая сумма

## Checkout
- [ ] Кнопка "Check out" переводит на страницу оформления заказа

## Пустая корзина
- [ ] При удалении всех товаров корзина становится пустой
- [ ] Отображается сообщение о пустой корзине (если есть)

## Навигация
- [ ] Кнопка "Continue shopping" возвращает в каталог
- [ ] Возможен переход в карточку товара из корзины

## UI проверки
- [ ] Изображения товаров отображаются корректно
- [ ] Верстка страницы не ломается
- [ ] Цена отображается в корректном формате
