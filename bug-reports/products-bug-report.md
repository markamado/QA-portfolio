# Bug Reports (SauceDemo)

### BUG-001: Не происходит переход в корзину после добавления товара и нажатия "My cart"

**Предусловие:**  
Пользователь находится на странице товара

**Шаги для воспроизведения:**
1. Открыть страницу товара (например, "Black heels")  
2. Нажать кнопку "Add to cart"  
3. Нажать на кнопку "My cart"  

**Фактический результат:**  
Страница корзины не открывается, отображается индикатор загрузки, переход не происходит  

**Ожидаемый результат:**  
Пользователь перенаправляется на страницу корзины с добавленным товаром  

**Severity:**  
High  

**Priority:**  
High
